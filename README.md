# Tech used

## Back end

- TypeScript
- Node
- Express
- MongoDB (Atlas)
- Hosted with Heroku (for now)

## Front end

- TypeScript
- React
- Redux
- PrimeReact UI Library
- Chart.js
- Luxon DateTime
- Sass

## Branch Structure

- `master`: usually the most recent prod deployment
- `dev`: latest code, may not be 100% functional
- `release-vx.x.x`: code for that particular release
